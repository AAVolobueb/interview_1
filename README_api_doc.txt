Эндпоинты api разбиты на 2 раздела: admin и user, для admin требуется BasicAuthentication, login: admin password: admin

Функционал для admin:
GET:
/api/v1/admin/interviews/ - возвращает полный список опросников, с вложенными в них вопросами и вариантами ответов.
/api/v1/admin/interviews/xx/ - где xx это id опросника, возвращает опросник, с вложенными в него вопросами и вариантами ответов.
/api/v1/admin/questions/xx/ - где xx это id вопроса, возвращает вопрос без вложенных вариантов ответов.
/api/v1/admin/answers/xx/ - где xx это id варианта ответа, возвращает вариант ответа.

POST:
/api/v1/admin/interviews/post/ - эндпоинт для добавления опросника с вложенными вопросами и вариантами ответов. Поля:

"interview_title": "" - Текст. Название опроса
"interview_description": "" - Текст. Описание опроса
"interview_end_date": "" - Дата и время окончания опроса
"question_set": [] - Список вопросов в опросе
"question_text": "" - Текст. Текст вопроса
"answer_type":  - Число. Тип вопроса, число: 1 - ответ текстом, 2 - выбор одного варианта ответа, 3 выбор нескольких вариантов ответа.
"answeroption_set": [] - Список вариантов ответа
"answer_text": "" - Текст. Текст варианта ответа

Пример JSON:

{
    "interview_title": "Название опроса",
    "interview_description": "Описание опроса",
    "interview_end_date": "2021-02-28T00:00:00Z",
    "question_set": [
        {
            "question_text": "текст вопроса 1",
            "answer_type": 1,
            "answeroption_set": []
        },
        {
            "question_text": "текст вопроса 2",
            "answer_type": 1,
            "answeroption_set": []
        },
        {
            "question_text": "текст вопроса 3",
            "answer_type": 2,
            "answeroption_set": [
                {
                    "answer_text": "Текст варианта ответа 1"
                },
                {
                    "answer_text": "Текст варианта ответа 2"
                },
                {
                    "answer_text": "Текст варианта ответа 3"
                },
                {
                    "answer_text": "Текст варианта ответа 4"
                }
            ]
        },
        {
            "question_text": "текст вопроса 4",
            "answer_type": 1,
            "answeroption_set": []
        }
    ]
}

DELETE:
/api/v1/admin/interviews/xx/ - где xx это id опросника, удаляет опросник, с вложенными в него вопросами и вариантами ответов, а также имеющимися для него результатами опросов от пользователей.
/api/v1/admin/questions/xx/ - где xx это id вопроса, удаляет вопрос с вложенными вариантами ответов.
/api/v1/admin/answers/xx/ - где xx это id варианта ответа, удаляет вариант ответа.

PATCH:
/api/v1/admin/interviews/xx/ - где xx это id опросника, меняет аттрибуты опросника и/или вложенных вопросов и/или вариантов ответов на указанные в JSON, добавляет вложенные вопросы и/или варинаты ответов. Пример JSON для изменения текста варианта ответа с id:92 вопроса с id:55 опросника с id:12. Если не указывать id вопроса или варианта ответа(или указать несуществующий id), то добавится новый.

{
    "id": 12,
    "question_set": [
        {
            "id": 55,
            "answeroption_set": [
                {
                    "id": 92,
                    "answer_text": "Измененный текст варианта ответа"
                }
            ]
        }
    ]
}


Функционал для user:
Авторизация не требуется
GET:
/api/v1/user/interviews/ - возвращает список опросников, для которых указанные дата и время окончания больше текущего времени.
/api/v1/user/interviews/xx/ - где xx это id опросника, возвращает опросник, с вложенными в него вопросами и вариантами ответов, если для него указанные дата и время окончания больше текущего времени.
/api/v1/user/results/xx/ - где xx это user_id пользователя, возвращает все сохраненные результаты опросов для указанного user_id.

POST:
/api/v1/user/results/ - Эндпоинт для добавления результатов опроса.
Поля:
"interview": - Число.  id опросника к которому сохраняется результат.
"user_id": - Число. user id под которым сохраняется результат опроса.
"resultanswer_set":[] - Список ответов.
"question_number": - Число. id вопроса к которому сохраняется ответ.
"answer": - Текст. Ответ пользователя на вопрос. Для вопросов типа 2(выбор варианта ответа) указывается id выбранного варианта ответа(число, но текстом, т.е. в кавычках). Для вопросов типа 3(выбор нескольких вариантов ответов) указываются id выбранных вариантов ответов через запятую.

Пример JSON:

{
    "interview": 70,
    "user_id": 15,
    "resultanswer_set":[
        {
            "question_number": 48,
            "answer": "Какой-то ответ"
        },
        {
            "question_number": 50,
            "answer": "81"
        },
        {
            "question_number": 52,
            "answer": "86,89,90"
        },
        {
            "question_number": 53,
            "answer": "Какой-то ответ 4"
        }
    ]
}
